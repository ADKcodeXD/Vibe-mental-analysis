**Self Mirror** is a deeply introspective AI-powered personality analysis engine. Unlike traditional MBTI tests, it combines clinical psychological screening (PHQ-9, MDQ, GAD-7), behavioral scenarios, and value assessments to generate a holographic profile of your true self.

### Features
- **Deep AI Analysis**: Uses LLM Chain of Thought (CoT) to analyze 50+ dimensions of your psyche.
- **Clinical Screening**: Includes professional-grade depression, anxiety, and bipolar assessments.
- **Truth Engine**: Automatically detects contradictions and deception in your answers.
- **Political Compass**: Analyzes your ideological leanings (e.g., Social Democracy, Right-Libertarianism).
- **Holographic Report**: Generates a stunning, shareable "Identity Card" with your archetype, RPG alignment, and shadow traits.

### Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open [http://localhost:3000](http://localhost:3000)** to start your journey.
